template<typename B, typename R>
class Decoder_SISO< B, R >

A Decoder_SISO is both a "Soft Input Hard Output" decoder and a "Soft Input Soft Output" decoder.

Rtype of the reals (floating-point or fixed-point representation) in the decoder.
