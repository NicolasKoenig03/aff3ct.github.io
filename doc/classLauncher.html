Detailed Description

template<typename B, typename R, typename Q>
class Launcher< B, R, Q >

Collects the command line parameters and launches the simulation. Describes and provides tools to make a working launcher.

The Simulation is an abstract and generic container.

Template Parameters
Btype of the bits in the simulation.
Rtype of the reals (floating-point representation) in the simulation.
Qtype of the quantified reals (fixed-point representation) in the simulation.
