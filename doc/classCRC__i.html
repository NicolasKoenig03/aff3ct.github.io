Detailed Description

template<typename B>
class CRC_i< B >

Adds/builds and checks a Cyclic Redundancy Check (CRC) for a set of information bits.

Template Parameters
Btype of the bits in the CRC.

Please use CRC for inheritance (instead of CRC_i).
