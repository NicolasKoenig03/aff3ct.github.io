Detailed Description

template<typename R>
class SISO_i< R >

A SISO (Soft Input Soft Output) is a type of decoder which takes a soft input and return a soft output.

Template Parameters
Rtype of the reals (floating-point or fixed-point representation) in the SISO.

Please use SISO for inheritance (instead of SISO_i).
