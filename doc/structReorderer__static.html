Detailed Description

template<typename T, int N_FRAMES>
class Reorderer_static< T, N_FRAMES >

Reorders a list of frames (the reordering code is static).

Template Parameters
Tthe type of data to reorder.
N_FRAMESthe number of frames to reorder (have to be a power of 2).
