State-of-the-Art Channel Coding Software Decoders

This page is dedicated to the State-of-the-Art Channel Coding Software Decoders. It allows to easily compare and see what it has been done and what you can expect from today software decoders. For now three big code families are considered : the Turbo codes (3G, 4G, CCSDS, etc.), the Low-Density Parity-Check (LDPC) codes (WiFi, WiMAX, CCSDS, WRAN, DVB-S2, etc.) and the new Polar codes.
