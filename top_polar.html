State-of-the-Art Channel Coding Software Decoders

This page is dedicated to the State-of-the-Art Channel Coding Software Decoders. It allows to easily compare and see what has been done and what can be expected from today software decoders. For now three big code families are considered : the Turbo codes (LTE, LTE-Advanced, CCSDS, etc.), the Low-Density Parity-Check (LDPC) codes (Wi-Fi, WiMAX, CCSDS, WRAN, DVB-S2, etc.) and the new Polar codes.
